## Required Verifications Before Phase 3

Before building any public-facing page in Phase 3, the following must be confirmed in writing:

### Priority 1 — Required for product pages
- [ ] Complete confirmed product list (series names, numbers, types)
- [ ] Frame material for each series (aluminum, uPVC, or both)
- [ ] Any performance certifications (AAMA, NFRC, etc.) with supporting documentation
- [ ] Title 24 compliance status with specific data or disclaimer
- [ ] Available hardware brands per series
- [ ] Available finish colors/codes per series
- [ ] Maximum/standard unit dimensions per series
- [ ] Lead time ranges (current, accurate)

### Priority 2 — Required for documentation / downloads section
- [ ] Confirm which downloadable documents actually exist (test reports, install guides, DWG files, BIM files)
- [ ] Confirm file formats, sizes, and update dates

### Priority 3 — Required before marketing copy is finalized
- [ ] Confirm "factory-direct" sourcing claim is accurate and legally defensible
- [ ] Confirm "European-grade" claim or remove
- [ ] Confirm geographic service area (Bay Area only? statewide? national?)
- [ ] Confirm any warranty terms
